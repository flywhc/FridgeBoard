# ADR-0003：媒体留存、调度、备份与运维

状态：已接受  
日期：2026-07-19

## 决策

拍照识别图片只作为一次 Agnes 请求的临时输入：完成、失败或超时即删除，最长 10 分钟。仅用户确认采用的 AI 小类图标会持久化；未选候选立即删除。

单容器调度器处理每日 Push、清理和软删除，所有可重复执行的工作都在 SQLite 留存幂等记录。服务器每日创建数据库一致性备份和持久图标资产快照、保留 7 天；升级前和至少每月手工复制备份至所有者本机。

## 原因

食品照片不属于首版产品数据，长期保留会增加隐私和存储负担。单调度器与本地备份可匹配当前服务器和家庭使用规模，避免外部队列、对象存储及监控平台的维护成本。

## 后果

- AI 请求所需的临时图片 URL 必须不可预测、短效，并禁止进入应用日志、备份和长期缓存。
- 服务器、容器、数据库和持久资产卷必须有清晰的备份与恢复操作；临时目录不在恢复范围。
- 自动部署前创建快照；若发布失败，不对已有数据执行自动破坏性回滚。
